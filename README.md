### Setup

```bash
# Cloning repo and installing dependencies
git clone https://github.com/gabrielrufino/RaspberryPocketServer
cd RaspberryPocketServer
npm install
```

Starting the server:
```bash
# For production
npm start

# For development
npm run dev
```
